LlamaIndex Newsletter 2025-04-22

  • Newsletter

Hi there, Llama Aficionados! 🦙

Welcome to this week's edition of the LlamaIndex newsletter! We're thrilled to share some exciting updates, including a deep research solution with Llama4, a new tutorial on building an equity research agent, and a community project showcasing a hierarchical multi-agent system.

Heads up: you can join our hands-on workshop in NYC on May 29th to learn how to build AI solutions for financial challenges. Apply to Attend.

🤩 The Highlights:

  • LlamaResearcher - Deep Research with Llama4: A fully open-source project by Clelia Bertelli that provides a complete deep research solution. It evaluates queries, expands them into sub-queries, searches the web, and produces essays based on the gathered information. Page, Repo.
  • Build an Equity Research Agent from Scratch: Our new tutorial shows you how to create an end-to-end agentic workflow that processes unstructured earnings reports for Tesla and Ford, extracting and normalizing financial metrics. Notebook.
  • Build a Hierarchical Multi-Agent System with LlamaIndex Supervisor: This community project demonstrates how to orchestrate multiple agents with a central supervisor, allowing for flexible task delegation and multi-level agent hierarchies. Repo.

🗺️ LlamaCloud & LlamaParse:

  • LlamaExtract for Financial Analysis: A full-stack JavaScript web app using LlamaExtract to perform financial analysis by extracting risk factors from corporate annual reports. Repo.
  • LlamaIndex Supports Gemini 2.5 Flash: Enjoy seamless integration with the new Gemini model without any upgrades required. Example.

✨ Framework:

  • A2A-Compatible Agents: Learn how to build agents that communicate securely and coordinate actions across different technologies using the A2A protocol. Example, Learn More.
  • Day 0 Support for GPT-4.1: OpenAI's latest model is now supported in our API, allowing you to enhance your applications with improved capabilities. Learn More.

✍️ Community:

  • Catch Our Founder at the AI User Conference: Don't miss @jerryjliu0 discussing how LlamaIndex can automate operational work for knowledge workers. Registration.
  • Building AI Agents with LlamaIndex: Explore how to create agents that auto-generate company risk reports and integrate with Google Cloud databases. Talk Deck, Video.

Thank you for being part of our Llama community! We look forward to bringing you more updates and innovations in the coming weeks.
