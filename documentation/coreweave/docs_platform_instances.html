Instances

About our hardware instances

CoreWeave offers an array of GPU and CPU-only hardware instances to support a range of compute-intensive workloads. Our instances are powered by NVIDIA GPUs and AMD and Intel CPUs.

Availability and Pricing​

See the Region reference for availability information and GPU instance pricing for hourly costs.

GPU instances​

Our GPU instances are designed for compute-intensive tasks that require significant parallel processing power. These instances are ideal for applications such as model training, inference, deep learning, high-performance computing (HPC), and other GPU-accelerated workloads. See the sections that follow for detailed information on each type.

GB200 NVL72-powered instances with InfiniBand​

Powered by four GB200 GPUs with InfiniBand connectivity, these instances represent the pinnacle of our high-performance computing offerings, with up to 1.44 exaFLOPS of AI compute power per rack. The GB200 GPUs are equipped with 30TB of high-bandwidth memory per rack and fifth-generation NVLink with up to 1 PB/s total bandwidth. These instances offer 4x faster training for trillion-parameter language models and 30x faster inference for large language models compared to previous-generation GPUs.

Choose these instances when you need maximum performance for large-scale AI training and inference, unprecedented memory capacity for massive datasets, and ultra-fast GPU-to-GPU communication for distributed computing. Instances must be requested in multiples of 18 to take advantage of the NVL72 architecture.

Detailed specifications:

  • Instance ID: gb200-4x
  • GPU Model: GB200
  • InfiniBand: Yes
  • GPU Count: 4
  • GPU RAM (GB): 186
  • CPU Model: 2x NVIDIA Grace (Arm v9)
  • vCPU Count: 144
  • RAM (GB): 960
  • Local Disk (TB): 30.72
  • Network Speed: Dual Port 100GbE

B200 with InfiniBand​

Powered by eight NVIDIA B200 Blackwell GPUs, each with 180 GB of HBM3e memory, these instances deliver outstanding performance for enterprise AI workloads, including training and real-time inference—offering more than twice the performance per GPU compared to previous-generation NVIDIA Hopper GPUs.

The GPUs are interconnected using NVIDIA NVLink and NVLink Switch to enable fast data transfer. Each instance also includes two Intel Emerald Rapids 8562Y+ CPUs running at 2.80 GHz and 61.44 TB of high-speed NVMe local storage. For networking, B200 instances feature an NVIDIA BlueField-3 DPU and eight NVIDIA ConnectX-7 InfiniBand host channel adapters (HCAs), supporting high throughput and lossless scaling. A 400G NDR non-blocking NVIDIA Quantum-2 InfiniBand fabric ensures seamless, high-speed connectivity.

Detailed specifications:

  • Instance ID: b200-8x
  • GPU Model: B200
  • InfiniBand: Yes
  • GPU Count: 8
  • GPU RAM (GB): 180
  • CPU Model: Intel Emerald Rapids (8562Y+, 2.80 GHz)
  • vCPU Count: 128
  • RAM (GB): 2048
  • Local Disk (TB): 61.44
  • Network Speed: Dual Port 100GbE
